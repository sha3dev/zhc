# IDEA

El objetivo es tener una organizacion de agentes, cada uno con:

    * Una personalidad (SOUL.md)
    * Un modelo asociado (GPT 5.4, Opus, etc)

Hay un agente que es el "CEO", y todos los demas agentes le reportan a el. Hay agentes que no
reportan al CEO directamente, sino a otros agentes.

La organizacion standard podria ser:

    * CEO
    * Product Designer
    * Frontend Developer
    * Backend Developer
    * Marketing Specialist
    * Customer Support Specialist

EL "human" (yo) hablo siempre con el CEO, el es quien habla con el resto de agentes.

El flujo es:

    * Yo le pido al CEO


Podemos crear "proyectos", estos proyectos no es mas que un flujo de tareas. Todo comienza
